# Pizza Delivery Application - Oasis Infobyte Internship

# Objective
To build a complete full stack Pizza Delivery Application with user 
and admin functionality, custom pizza builder, inventory management, 
order tracking and payment integration.

## Steps Performed
1. Designed the database models for Users, Orders and Inventory
2. Built REST APIs using Node.js and Express.js
3. Implemented JWT authentication for User and Admin roles
4. Built the React frontend with a dark pizza themed UI
5. Created a 5-step custom pizza builder
6. Built admin dashboard for inventory and order management
7. Implemented real-time order status tracking
8. Integrated payment gateway for order placement
9. Added email notifications for order status updates
10. Tested the complete application end to end

## Tools Used
- HTML
- CSS
- JavaScript
- React.js (Frontend)
- Node.js (Backend)
- Express.js (REST APIs)
- MongoDB (Database)
- JWT (Authentication)
- Nodemailer (Email Notifications)
- VS Code (Code Editor)
- Chrome Browser (Testing)
- MongoDB Compass (Database Management)
- GitHub (Version Control)

## Outcome
Successfully built a complete full stack pizza delivery application with:
- User registration and login system
- Admin and User role based access
- 5-step custom pizza builder
- Choose from 5 bases, 5 sauces, 5 cheeses and 10 veggies
- Admin inventory management with stock tracking
- Low stock email alerts for admin
- Real-time order status tracking
- Order status: Order Received → In the Kitchen → Sent to Delivery → Delivered
- Payment integration for order placement
- Email notifications for every order status update
